# Movies

Тестовое задание - Приложение для поиска и просмотра информации о фильмах.

## Функционал

- Просмотр списка популярных фильмов на главной странице
- Поиск фильмов и сериалов по названию
- Продвинутая фильтрация (жанры, страны, года, рейтинги, провайдеры)
- Просмотр детальной информации: рейтинги (KP, IMDB), описание, актеры, похожие фильмы
- Сравнение выбранных фильмов
- Сохранение фильмов в список "Избранное" (с использованием LocalStorage)
- Переключение темы оформления
- Переключение языков (RU / EN)
- Виртуализация длинных списков
- Оптимизированные скелетоны для плавности интерфейса при загрузке
- Адаптивная верстка (VKUI)

## Используемые технологии

- React 19
- TypeScript
- Vite (сборка проекта)
- VKUI (библиотека компонентов)
- React Router (маршрутизация)
- TanStack Query (управление состоянием и кэширование)
- i18next (интернационализация)
- Sass (CSS-модули)
- Axios
- React Virtuoso (виртуализация списков)

## Установка и запуск проекта

- Клонировать репозиторий: `git clone https://github.com/sdlmdev/movies`
- Установить зависимости: `npm install`
- Запустить проект: `npm run start` или `npm run dev`
